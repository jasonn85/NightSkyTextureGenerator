# NightSkyTextureGenerator
A MacOS app to generate a cube texture set of visible stars using the Yale Bright Star Catalog.

## Supported file formats
- PNG
- TIFF

## Features
- Adjustable resolution from 256 to 32k
- Magnitude filtering

# Resources
- Yale Bright Star Catalog (http://tdc-www.harvard.edu/catalogs/bsc5.html)
- https://jorenjoestar.github.io/post/realistic_stars/realistic_stars/

- Single-pass Rendering of Day and Night Sky Phenomena (https://www.researchgate.net/publication/287031803_Single-pass_Rendering_of_Day_and_Night_Sky_Phenomena)
